# Prompt 02 — Build

> This prompt is responsible for building out the core features and UI of the UNPRO application.

## What This Prompt Generates

- **Public Pages** — Landing page, contractor directory, about/pricing pages
- **Dashboard** — Homeowner dashboard with property overview, quotes, scores
- **Quote Analyzer** — Upload and AI-analyze contractor quotes for fairness
- **Home Score** — Property condition scoring interface with knowledge graph
- **AIPP Score** — AI-powered performance scoring display and trends
- **Contractor Profiles** — Public profile pages, portfolio, reviews
- **Pro Features** — Premium contractor tools, analytics, lead management
- **Admin Panel** — User management, verification queue, platform analytics

## Prerequisites

- Prompt 01 (Foundation) must be completed
- Database schema and auth must be functional

## Expected Output

After running this prompt, the application should have:
1. All pages fully built with real UI components
2. Data fetching connected to Lovable Cloud
3. Forms with validation and error handling
4. Responsive design across all breakpoints
5. Navigation and routing fully functional
