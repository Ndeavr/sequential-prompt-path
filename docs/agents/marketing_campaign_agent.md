# UNPRO Marketing Campaign Agent

This agent determines which marketing campaigns should be launched.

## Inputs

- Demand signals
- Weather events
- Seasonal services
- Professional supply gaps

## Examples

### Spring Campaigns
- Patio construction
- Landscaping
- Fencing

### Fall Campaigns
- Insulation
- Window sealing

## Outputs

- Marketing campaign calendar
- Landing page ideas
- SEO expansion recommendations
