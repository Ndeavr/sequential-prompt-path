# UNPRO Weather Impact Agent

The Weather Impact Agent monitors weather events and predicts resulting homeowner service demand.

## Signals Monitored

- Heavy rainfall
- Flooding
- Wind storms
- Snow accumulation
- Extreme cold

## Example Mappings

| Weather Event | Service Demand |
|---------------|---------------|
| Heavy rain | Drainage services |
| Storm winds | Roofing repair |
| Cold waves | Insulation demand |

## Outputs

- Demand spike alerts
- Service recommendations
- Contractor notifications
