# UNPRO — Property Knowledge Graph

> This document describes the property knowledge graph — a structured data model that captures everything known about a property over time.

## Purpose

The Property Knowledge Graph aggregates inspection data, maintenance history, contractor work, quotes, and scoring into a unified view per property.

- **Timeline View** — Chronological history of all property events
